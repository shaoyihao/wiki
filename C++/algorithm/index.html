    next_permutation 与 prev_permutation¶

    在 C++ 中，一个序列所有元素所构成的 \(n!\) 个排列之间是唯一有序的，称为字典序（基准排序序列），例如：

    • abc 所有元素构成的基准排列序列为：abc，acb，bac，bca，cab，cba
    • 123 所有元素构成的基准排列序列为：123，132，213，231，312，321

    对于一个数组 a 执行 next_permutation(a + i, a + j) 函数，结果为数组 a 发生了改变：数组中的子段 \(a[i\sim j - 1]\) 的基准排列序列中的下一个序列取代了该子段。 若转化的序列的字典序更大，则返回 true，若转化的序列的字典序变小（循环到开头），则返回 false。

    prev_permutation 与 next_permutation 的使用方法相同：

    next_permutation(first, last);  // 转化为下一个排列
prev_permutation(first, last);  // 转化为上一个排列

    三个参数¶

    find¶

    find(first, last, x);   // 在[first, last)中顺序寻找x
    • 若找到，则返回指向该元素（首次出现）的迭代器；
    • 若未找到，则返回 \(\text{last}\)

    排序¶

    sort(first, last, cmp);
stable_sort(first, last, cmp);  // 稳定排序
    cmp谓词
    • 自己写比较函数 
      bool cmp(int a, int b)
{
    return a < b; 
}
    • 使用 Lambda 函数： 
      [](int a, int b)->bool{return a > b;}
    • 使用 functional 标准库：该标准库提供了一些基于模板的比较函数对象：
      • equal_to<T>、not_equal_to<T>
      • greater<T>、greater_equal<T>、less<T>、less_equal<T>

    lower_bound 与 upper_bound¶

    lower_bound(first, last, x);    // 二分查找第一个 ≥x 的元素
upper_bound(first, last, x);    // 二分查找第一个 >x 的元素
    返回元素下标： 
    int pos = lower_bound(a, a + n, x) - a;

    四个参数¶

    nth_element(first, first + k, last, cmp);  // 划分(partition)
    • nth_element(first, first + k, last) ：将第 \(k\) 小的元素放到第 \(k\) 个位置上，并且 左部分 \(\le\) 右部分
    • nth_element(first, first + k, last, greater<int>()) ：将第 \(k\) 大的元素放到第 \(k\) 个位置上，并且 左部分 \(\ge\) 右部分
