    bitset

    bitset 用于创建二进制串。

    构造函数¶

    1
2
3
    bitset<n> b;           // b有n位，每位都为0
bitset<32> b(x);       // 由一个32位数构造bitset
bitset<5> b("11001");  // 直接由字符串构造bitset   （下标最大的字符称为最低比特）

    基本操作¶

    • ~、&、|、^、>>、<<、==、!=

    • []：访问某一位（最右侧为第 0 位），可以进行赋值

    • 其它操作：

      • count()：统计有多少位是 1

      • set()：将所有位置 1

        reset()：将所有位置 0

        flip()：将所有位取反

      • any()：若至少有一位是 1，则返回 true

        none()：若全是 0，则返回 true

      • to_ulong()：转换为 unsigned long 类型的值
