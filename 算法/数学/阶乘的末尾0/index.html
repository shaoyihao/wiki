    阶乘的末尾0

    题目：给定一个正整数 \(n\)，求 \(n!\) 末尾有多少个 \(0\)？

    分析：一个数末尾的 \(0\) 来自于因数 \(2\) 和 \(5\)，例如：（设 \(x\) 不含因子 \(2,5\)）

    • \(x\times 2\times 5=x\times10\)：末尾有 \(1\) 个 \(0\)
    • \(x\times 2\times 2\times 5\times 5=x\times100\)：末尾有 \(2\) 个 \(0\)
    • \(x\times \underbrace{2\times\cdots\times 2}_{m个}\times \underbrace{5\times\cdots\times 5}_{n个}=x'\times10^{\min\{m,n\}}\)：末尾有 \(\min\{m,n\}\) 个 \(0\)

    因此问题转化为求 \(n!\) 可分解出多少个因子 \(2,5\)​。

    • 如何求出 \(n!\) 中包含多少个质因子 \(p\) ？

      \(1\sim n\) 中能被 \(p\) 整除的数（即 \(p\) 的倍数）的个数为 \(\lfloor\dfrac{n}{p}\rfloor\)，因此 \(n!\) 中因子 \(p\) 的个数为：

      \[ \lfloor\frac{n}{p}\rfloor+\lfloor\frac{n}{p^2}\rfloor+\lfloor\frac{n}{p^3}\rfloor+\cdots \]

      int cnt = 0, x = p;
while (x <= n)
{
    cnt += n / x;
    x *= p;
}

    • \(n!\) 分解出的 \(2\) 的数目大于 \(5\) 的数目。

      证明

      设 \(2,5\) 的数目分别为 \(a,b\)，则：

      \[ \begin{align} a=\lfloor\frac{n}{2}\rfloor+\lfloor\frac{n}{2^2}\rfloor+\lfloor\frac{n}{2^3}\rfloor+\cdots \tag{1}\\ b=\lfloor\frac{n}{5}\rfloor+\lfloor\frac{n}{5^2}\rfloor+\lfloor\frac{n}{5^3}\rfloor+\cdots\tag{2} \end{align} \]

      \((1)\) 式中的每一项都大于 \((2)\) 式中的每一项，且 \((1)\) 式项数更多，因此 \(a>b\)。

      因此本题转化为求 \(n!\) 中可分解出的 \(5\) 的数目。

    #include <iostream>
using namespace std;

int main()
{
    int n;
    cin >> n;

    int cnt = 0, t = 5;
    while (t <= n) cnt += n / t, t *= 5;
    cout << cnt;

    return 0;
}
