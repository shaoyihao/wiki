    由遍历序列建立¶

    本质上即根据先序和中序、后序和中序、... 等信息对二叉树进行遍历。

    结论

    • 先序、后序、层序序列只知道其中一种、或两两搭配、或三个一起都无法构建唯一的二叉树。原因：先序、后序、层序均是提供根结点，作用是相同的，都必须由中序序列来区分出左右子树。
    • 只有同时知道先序和中序、或同时知道后序和中序、或同时知道层序和中序，才能唯一确定出原二叉树。
    • 给定一个先序序列也就给定了各结点的入栈顺序，可以产生 \(\dfrac{\mathsf{C}_{2n}^n}{n+1}\) 不同的出栈顺序（中序序列），而一组先序、中序序列可以唯一确定出一个二叉树，因此一个先序序列对应二叉树有 \(\boxed{\dfrac{\mathsf{C}_{2n}^n}{n+1}}\) 个。

    • 仅由一个先序序列无法唯一确定一个二叉树，但若用字符表示出所有空元素，所形成的二叉树则是唯一的。

      void CreatBiTree(BiTree &T)
{
    ElemType ch;
    cin >> ch;
    if (ch == '#') T = NULL;    // ‘#’ 号即代表着空
    else
    {
        T = new BiNode;
        T->data = ch;           // 建立根结点
        CreatBiTree(T->Lchild); // 构造左子树
        CreatBiTree(T->Rchild); // 构造右子树
    }
}

      // 由长度为len的先序序列和中序序列创建一个二叉树
void Create1(BiTree &T, char *pre, char *in, int len)
{
    if (len <= 0) { T = NULL; return; }

    T = new BiNode;
    T->data = pre[0]; // 先序序列的第一个元素即根结点

    // 在中序序列中查找根结点（temp记录所遍历到的字符至结尾的数组）
    char *temp;
    for (temp = in; temp < in + len; temp++)
        if (*temp == pre[0]) break;
    int k = temp - in; // k即左子树的长度

    Create1(T->Lchild, pre + 1, in, k);                     // 建立左子树
    Create1(T->Rchild, pre + 1 + k, temp + 1, len - 1 - k); // 建立右子树
}

      void Create2(BiTree &T, char *in, char *post, int len)
{
    if (len <= 0) { T = NULL; return; }

    T = new BiNode;
    T->data = post[len - 1];

    char *temp;
    for (temp = in; temp < in + len; temp++)
        if (*temp == post[len - 1]) break;
    int k = temp - in;  // 左子树的长度

    Create2(T->Lchild, in, post, k);
    Create2(T->Rchild, temp + 1, post + k, len - 1 - k);
}

    • 知道先序和后序无法唯一地还原出原二叉树。原因：如果一个子树的根结点只有单个子结点，那么无论该结点是左孩子还是右孩子，该子树的前序和后序序列都是一样的，使得不同的树可能对应相同的“前序+后序”组合。例如：

      注意

      如果二叉树的每个非叶结点都有两个孩子结点，则由前序、后序序列可以唯一确定这棵二叉树。

      虽无法还原出二叉树，但可以得出树中结点间的父子关系：

      由先序和后序序列还原二叉树的方法：（不妨设数组从下标 1 开始存储）

      • 先序序列的首个元素（后序序列的尾元素）即根结点。

      • 寻找先序序列的第 2 个元素在后序序列中的位置（设为 \(j\)）。

      • 若 \(j=n-1\)，则说明根结点只有一个子树，可以是左子树，也可以是右子树。

        否则 \(\text{post}[1,j]\)（即 \(\text{pre}[2,2+j)\) ）构成了左子树，右子树所含元素也随之确定。

      • 对于其余子树，可采用类似的方法递归处理。

      特别的，若先序序列和后序序列顺序（长度为 \(n\)）完全相反，则说明二叉树中每个结点都只有一个子树，即二叉树的高度为 \(n\)，有 \(2^{n-1}\)​ 种不同的树。

      例子

      已知先序序列 1 2 4 5 6 3，后序序列 4 2 3 6 5 1，则：

      1. 根结点：1
      2. 寻找 2 在后序序列中的位置，可得 4 2 为左子树中的元素，3 6 5 为右子树中的元素
        • 考虑左子树（\(n=2\)），先序序列 2 4，后序序列 4 2，顺序完全相反，因此共有 \(2^{2-1}=2\) 种不同的子树
        • 考虑右子树（\(n=3\)），先序序列 5 6 3，后序序列 3 6 5，顺序完全相反，因此共有 \(2^{3-1}=4\) 种不同的子树
      3. 因此共有 \(2\times4=8\) 种不同的二叉树
