    棋盘游戏

    题目：给定一个 \(6\times 6\) 的棋盘，棋盘的每个位置上都有一个数值，求从起始位置到终止位置的最小总代价。规定：

    • 初始状态为 1
    • 只能沿上下左右四个方向移动
    • 若从 \((a,b)\)（状态为 \(\text{st}\)）移动到 \((c,d)\)，则：
      • 该步的代价 \(\text{cost}=\text{st}\times g[c][d]\)
      • 新状态 \(\text{st}'=\text{cost}\ \%\ 4+1\)​
    • 总代价是路径中所有步的代价之和

    分析：将 “坐标 + 状态” 视为一个整体：

    struct Node
{
    int x, y, s;
};

    从结点 p 移动到结点 q 的代价 cost = p.s * g[q.x][q.y]，且 q.s = cost % 4 + 1。

    如此我们便定义出了“结点”和“结点间的距离”，起点即 {sx, sy, 1}，终点即 {ex, ey, i}，转化为最短路问题。

    int dist[N][N][5];  // {x, y, s} → dist
bool st[N][N][5];   // {x, y, s} → st

void spfa()
{
    memset(dist, 0x3f, sizeof(dist));

    queue<Node> q;
    q.push({sx, sy, 1});
    dist[sx][sy][1] = 0;

    while (!q.empty())
    {
        Node t = q.front(); q.pop();
        st[t.x][t.y][t.s] = false;

        for (int i = 0; i < 4; i++) // 遍历邻接点
        {
            int x = t.x + dx[i], y = t.y + dy[i];  // 从 t 到 {x, y, s}
            if (x >= 0 && x < 6 && y >= 0 && y < 6)
            {
                int cost = t.s * g[x][y];
                int s = cost % 4 + 1;

                if (dist[x][y][s] > dist[t.x][t.y][t.s] + cost)  
                {
                    dist[x][y][s] = dist[t.x][t.y][t.s] + cost;
                    if (!st[x][y][s])
                    {
                        q.push({x, y, s});
                        st[x][y][s] = true;
                    }
                }
            }
        }
    }
}

    最终答案即 \(\min\limits_i\{\text{dist}[ex][ey][i]\}\) 。
