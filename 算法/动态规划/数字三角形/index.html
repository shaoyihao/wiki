    数字三角形

    题目：给定一个数字三角形，从顶部出发，在每一结点可以选择移动至其左下方的结点或移动至其右下方的结点，一直走到底层，要求找出一条路径，使路径上的数字的和最大。

    分析：对数字三角形中的每一个数字进行编号。

    令 \(f(i,j)\) 的值为：所有从起点走到 \((i,j)\) 的路径所对应的数字之和的最大值。求 \(f(i,j)\)​ 可以转化为求左上方的最大值和右上方的最大值（考虑最后一步的走法）：

    \[ f(i,j)=\max\{f(i-1,j-1),f(i-1,j)\}+a_{i,j} \]

    \(f(i,j)\) 仅与前面的状态有关，因此可以通过线性递推求解。

    初始化时，要注意初始化的范围。

    变式2¶

    题目：\(r\times c\) 的网格中放置着正整数，从左上角到右下角选择两条路径（只能向右或向下）使得总和最大。注意：若某个格子被一条路径选中，则该格子中的数变成 \(0\)。

    错误的思路

    变式 1 中可以求出从左上角到右下角和最大的路径，本题是否可以先求出一条和最大的路径，再在剩下的图中求另一条和最大的路径，二者相加即认为是最优解？

    这种贪心思路是错误的。考虑下面这个图：

    如果分两次，第一次最优肯定是把三个 2 取到然后结束，第二次就只能取一个 1。但实际上最优解可以将图中所有数都取到。

    若分两次，则第一次的最优解会影响到第二次的最优解，两次局部最优无法得到全局最优。因此本题应从“两条路径同时走”的角度来考虑。

    考虑两个人 A,B 同时从左上角出发，走出 2 条路径，最终同时到达终点。令 \(f(x_1,y_1,x_2,y_2)\) 表示“ A 从起点走到 \((x_1,y_1)\)、B 从起点走到 \((x_2,y_2)\) ”所有方案中的最大值。

    注意到，在任意时刻 A,B 所走的步数相同，满足 \(x_1+y_1=x_2+y_2\triangleq k\)，因此本题可以只用 \(3\) 个变量来表示一个状态：\(f(k, x_1,x_2)\)。最终答案即 \(f(r+c,r,r)\) 。

    前一步有 \(4\)（\(2\times 2\)）种情况（不一定都存在）：

    • A 向下走，B 向下走：\(f(x_1-1,y_1,x_2-1,y_2)\)，即 \(f(k-1,x_1-1,x_2-1)\)
    • A 向下走，B 向右走：\(f(x_1-1,y_1,x_2,y_2-1)\)，即 \(f(k-1,x_1-1,x_2)\)
    • A 向右走，B 向下走：\(f(x_1,y_1-1,x_2-1,y_2)\)，即 \(f(k-1,x_1,x_2-1)\)
    • A 向右走，B 向右走：\(f(x_1,y_1-1,x_2,y_2-1)\)，即 \(f(k-1,x_1,x_2)\)

    因此：

    \[ f(k,x_1,x_2)=\max\{f(k-1,x_1-1,x_2-1),f(k-1,x_1-1,x_2),f(k-1,x_1,x_2-1),f(k-1,x_1,x_2)\}+t \]

    增量 \(t\) 与 A,B 是否走到相同的格子有关：

    • 若 \((x_1,y_1)\ne (x_2,y_2)\)，则 \(t=w[x_1][y_1]+w[x_2][y_2]\)
    • 若 \((x_1,y_1)=(x_2,y_2)\)，说明 A,B 走到了相同的格子，则这个格子上的值只会被加一次，即 \(t=w[x_1][y_1]\)（\(=w[x_2][y_2]\)）

    for (int x1 = 1; x1 <= r; x1++)

            int y1 = k - x1, y2 = k - x2;                 // (1,1)→(x1,y1), (1,1)→(x2,y2)
            if (y1 >= 1 && y1 <= c && y2 >= 1 && y2 <= c) // 合法状态
            {
                int t;
                if (x1 == x2) t = w[x1][y1];
                else t = w[x1][y1] + w[x2][y2];

                f[k][x1][x2] = f[k - 1][x1][x2] + t;
                if (x1 > 1)           f[k][x1][x2] = max(f[k][x1][x2], f[k-1][x1-1][x2]   + t);
                if (x2 > 1)           f[k][x1][x2] = max(f[k][x1][x2], f[k-1][x1][x2-1]   + t);
                if (x1 > 1 && x2 > 1) f[k][x1][x2] = max(f[k][x1][x2], f[k-1][x1-1][x2-1] + t);
            }
        }
