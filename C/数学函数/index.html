    数学函数

    函数 功能
    abs、fabs 取绝对值
    floor、ceil 取整
    round 四舍五入
    pow 幂
    sqrt 开平方
    log 以 \(e\) 为底的对数
    sin、cos、tan、asin、acos、atan 三角函数

    一些值得注意的点

    • C 语言中没有对任意底数求对数的函数，因此必须使用换底公式进行转换：

      \[ \log_ba=\dfrac{\ln a}{\ln b} \]

    • \(\pi\) 的精确值可用 acos(-1) 进行计算
